gitphp-theme
============

a more modern theme for [gitphp](http://www.gitphp.org/)


##installation

copy gitweb.css to path/to/gitphp/css

modify path/to/gitphp/config/gitphp.conf.php, add following config  
```
$gitphp_conf['stylesheet'] = 'gitweb.css';
```
